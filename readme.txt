== Description ==
### INTRODUCTION
Price Meter Products Sync is official [Price Meter](https://pricemeter.pk) plugin for wordpress users to allow them easily sync their products on Price Meter without much effort.
This plugin uses [Price Meter PHP library](https://github.com/ict-master-pk/pricemeter-products-php) for all API calls.

### REQUIREMENTS
* Price Meter store import API key

### FEATURES
* Update product keywords right through your product's General tab
* One click sync, let you create or update your product on Price Meter
* Sync status indicator show product sync status on Products table
* With auto sync, whenever you create, update or delete your product it automatically update your product on Price Meter without any further action
* For bulk update your products, just download CSV right from plugin's setting page with one click and upload it on Price Meter

== Installation ==
### INSTALLATION
* Go to plugins page and click Add New in your WordPress Dashboard
* Click the upload tab
* Browse for the plugin file (pricemeter-products-sync.zip) on your computer
* Click “Install Now” and then hit the activate button
